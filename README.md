# 概要

[riot](https://github.com/riot/riot) を使ってシングルページアプリケーションを構築するためのテンプレートです。

## Usage

置くだけて使えます。

## Functions

- Router によるページ切り替え
- グローバルナビゲーション
- WEB-API のコール
- 簡単なReduxの利用
